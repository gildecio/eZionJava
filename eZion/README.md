# eZion Spring Boot API

This is a basic Spring Boot API project.

## Running the Application

To run the application, use:

```bash
mvn spring-boot:run
```

The API will be available at http://localhost:8080

## Endpoints

- GET /hello : Returns a hello message

## Running Tests

To run the tests:

```bash
mvn test
```

## Building

To build the project:

```bash
mvn clean package
```
